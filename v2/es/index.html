DjValidator es el plugin de jquery para validación de formularios web más simple, flexible y rápido de usar.

DjValidator 2: Más simple, más potente...


Características:

  • Solo requiere Jquery, 1.5.1 o superior, no depende de ningún UI framework.
  • Las reglas de validación se colocan usando un único atributo "data-dj-validator", además reconoce el atributo “required” y valida el campo como obligatorio.
  • Tiene todas las opciones de personalización.
  • Juego de reglas de validación multifuncionales compacto y de escritura rápida.
  • Diferentes formas de ejecutar la validación.
  • Soporta validación en servidor (asíncrona).
  • 22 reglas de validación disponibles.
  • Se puede crear validadores propios mediante 3 alternativas diferentes.
  • Se pueden cambiar las reglas de validación de forma dinámica.
  • Capacidad de cambiar el estilo de los mensajes de validación.

Ejemplo:

Uso Básico:

  1. Agregue los scripts de jquery and djValidator: 
    	    		
<script type="text/javascript" src="../lib/jquery/jquery.js"></script>
<script type="text/javascript" src="../dist/DjValidator.js"></script>

    Descargar desde el repositorio

    La librería también está disponible en Npm y Bower.

    	    		
npm i djvalidator

  2. Coloque las reglas de validación que necesite: 
    	    		
<input name="input1" type="text" data-dj-validator="atext,3,12" required>

    Agregue la regla de validación en los campos necesarios con la propiedad data-dj-validator, use el atributo required cuando el campo sea obligatorio:

    Revise aquí todos los validadores disponibles.


  3. Use el plugin en alguno de estos modos:

    Submit:La validación se ejecuta en el evento submit del formulario, realizando el envío automático cuando el formulario es válido, también sirve para validar campos, es el valor por defecto:

    	    		
//por defecto el modo es "submit"
$('#form').djValidator();
//Validando un solo campo
var resp=$('#input').djValidator();

    Function:La validación se ejecuta inmediatamente como una función, devuelve true o false según el resultado:

    	    		
var resp=$('#form').djValidator({mode:’function’});
